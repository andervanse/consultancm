O que é NCM?

A Nomenclatura Comum do Mercosul (NCM) é um código de oito dígitos usado para identificar a natureza dos produtos comercializados no Brasil e nos outros países do Mercosul, esta nomeclatura foi adotada em janeiro de 1995. Estes códigos utilizam o método internacional de classificação conhecido como SH (Sistema Harmonizado de Designação e Codificação de Mercadorias)

O que é o código NCM de uma mercadoria?

A NCM contém pouco mais de 10.000 códigos, sendo que seus códigos de 8 dígitos são chamados comumente de "Códigos NCM", pois são os que definem as alíquotas de impostos no comércio exterior e de diversos tributos internos nas operações com mercadorias, entre outras utilizações.

O Código NCM é composto por 8 posições sendo elas

  • 00 00 00 0 0  Capítulo     (2 primeiros dígitos da SH)
  • 00 00 00 0 0  Posição     (4 primeiros dígitos da SH)
  • 00 00 00 0 0  Subposição  (6 primeiros dígitos da SH)
  • 00 00 00 0 0  Item        (7º dígito da NCM)
  • 00 00 00 0 0  Subitem     (8º dígito da NCM)

fonte: receita.economia.gov.br
